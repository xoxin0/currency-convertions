# Диаграмма последовательности: конвертация валюты

## Бизнес-процесс

Пользователь хочет конвертировать 100 USD в EUR.

## Сценарий 1: прямой курс (курс USD→EUR существует)

| Клиент API | БД Калькулятор |              |               |
|------------|----------------|--------------|---------------|
| GET /exchange |             |              |               |
| ?from=USD  |                |              |               |
| &to=EUR    |                |              |               |
| &amount=100|                |              |               |
|------------>|               |              |               |
|            |                |              |               |
|            | 1. Запрос курса USD→EUR |   |               |
|            |---------------->|             |               |
|            |                 |             |               |
|            | 2. Курс = 0.92  |             |               |
|            |<----------------|             |               |
|            |                 |             |               |
|            | 3. 100 * 0.92 = 92.00 |     |               |
|            |---------------->|             |               |
|            |                 |             |               |
| 4. Response|                 |             |               |
| {converted: 92.00} |         |             |               |
|<-----------|                 |             |               |

## Сценарий 2: обратный курс (есть EUR→USD, нет USD→EUR)

| Клиент API | БД Калькулятор |              |               |
|------------|----------------|--------------|---------------|
| GET /exchange |             |              |               |
| ?from=USD  |                |              |               |
| &to=EUR    |                |              |               |
| &amount=100|                |              |               |
|------------>|               |              |               |
|            |                |              |               |
|            | 1. Ищем USD→EUR |             |               |
|            |---------------->|             |               |
|            |                 |             |               |
|            | 2. Не найден (404 logic) |  |               |
|            |<----------------|             |               |
|            |                 |             |               |
|            | 3. Ищем EUR→USD |             |               |
|            |---------------->|             |               |
|            |                 |             |               |
|            | 4. Курс = 1.086956 |        |               |
|            |<----------------|             |               |
|            |                 |             |               |
|            | 5. Обратный курс 1/1.086 = 0.92 | |         |
|            |---------------->|             |               |
|            |                 |             |               |
|            | 6. 100 * 0.92 = 92.00 |     |               |
|            |---------------->|             |               |

## Сценарий 3: кросс-курс через USD (есть USD→EUR и USD→RUB, нужно EUR→RUB)

| Клиент API | БД Калькулятор |              |               |
|------------|----------------|--------------|---------------|
| GET /exchange |             |              |               |
| ?from=EUR  |                |              |               |

| &amount=100|                |              |               |
|------------>|               |              |               |
|            |                |              |               |
|            | 1. Ищем EUR→RUB |             |               |
|            |---------------->|             |               |
|            |                 |             |               |
|            | 2. Не найден    |             |               |
|            |<----------------|             |               |
|            |                 |             |               |
|            | 3. Ищем EUR→USD |             |               |
|            |---------------->|             |               |
|            |                 |             |               |
|            | 4. курс: 1.086  |             |               |

|            |---------------->|             |               |

## Важные замечания

1. **Все три сценария должны быть реализованы** в модуле `calculator.py`
2. **Если ни один сценарий не подошёл** → возвращать 404 с сообщением "Курс не найден"
3. **Округление** итоговой суммы до 2 знаков (для валют)
